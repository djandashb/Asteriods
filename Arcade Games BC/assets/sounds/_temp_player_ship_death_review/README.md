# Player Ship Death Sound Review

Temporary review sound for the player ship being destroyed.

Generated asset:

- `player_ship_destroyed.wav`: 44.1kHz, 16-bit, stereo PCM WAV

Regenerate after tweaks:

```zsh
python3 assets/sounds/_temp_player_ship_death_review/generate_player_ship_destroyed.py
```

Design notes:

- Inspired by the original Atari Asteroids discrete explosion/noise circuit behavior.
- Uses a sharp noise burst, short crack, low falling rumble, and a fading metallic edge.
- This is an original synthesized sound, not a copied sample.
