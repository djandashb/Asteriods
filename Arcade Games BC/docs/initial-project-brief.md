# Initial Project Brief

## Project name

Retro Arcade Collection

## Primary machine

MacBook Pro M2

## Development approach

Build the project on macOS first using VS Code, Codex, C++ and SDL3.

## First game

Asteroids-style game based on an existing Python/Pygame reference version.

## Main principle

Rebuild cleanly in C++/SDL3. Do not perform a line-by-line Python translation.

## Main targets

- macOS playable version
- Windows 10/11 playable version
- Future iPhone/iPad version
- Future commercial collection

## First milestone

A clean C++ SDL3 Asteroids-style game running on macOS with:

- Player ship
- Rotation
- Thrust
- Shooting
- Asteroids
- Asteroid splitting
- Collision
- Score
- Lives
- Game over
